Code couleur :

Bureau non vide

Bureau surpeuplé

Liste des bureaux :

Nom : Le nom du bureau.

Capacité Max : le nombre de place dans un bureaux.

Pourcentage : Le pourcentage d'occupation du bureaux.

: Permet d'afficher le détaille du bureau.

Liste des personnes :

Nom : Le nom de la personne.

Prénom : Le prénom de la personne.

Status : Le Status de la personne.

Équipe : L'équipe de la personne.

Département : Le Département de la personne.

Bureau : Le bureaux de la personne.

Plan des bureaux :

Bureau
Salle, Mur, Détail
Escalier

Filtre :

Bureau :

Bâtiment : Il est possible de filtrer les bureaux selon le bâtiment ( C'est-à-dire : Loria, bâtiment A, bâtiment B, bâtiment C ). Puis les filtrer par étage.

État : Il est possible de filtrer les bureaux selon l'état du bureau. C'est-à-dire si le bureau est vide, non plein, plein, surpeuplé ou avec un ou plusieurs intrus.

Nom : Il est possible de filtrer les bureaux selon leur nom. C'est-à-dire qu'ils vont afficher que les bureaux qui commencent de la même manière.

Personne : Il est possible de retrouver le bureau d'une personne.

Date : Il est possible d'afficher les bureaux pour une date donnée. C'est-à-dire qu'ils vont afficher les bureaux avec des capacités peu être différente.

Personne :

Département : Il est possible de filtrer les personnes selon le département ( Par exemple : Informatique, Math, Biologie... ). Puis les filtrer par équipe.

Statut : Il est possible de filtrer les personnes selon le statut de la personne. C'est-à-dire Doctorant, Stagiare, Enseignant-chercheur ...

Nom : Il est possible de filtrer les personnes selon leur nom. C'est-à-dire qu'ils vont afficher que les personnes dont le nom ou prénom qui commence de la même manière.

Statut :

Enseignant-chercheur / Enseignante-chercheuse

Doctorant / Doctorante
